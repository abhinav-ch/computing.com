Software

Software is a set of instructions in an order that let us interact with the computer.
Software can not be removed by taking out a part of the computer.
The hard disk stores the sofware in the form of binary.
An OS is a type of software. Apps, OS, Programs are all software.

How software allows to interact with the computer hardware

The software takes the user's input and sends it to the os with an instruction.
The OS is also a type of software and uses the instructions given by the program to process the input and Sends it to the Computer hardware.
The computer hardware understands the processed input which the os gives and the computer hardware executes it. (Diagram explains this paragraph)

Types of Sofware

  • System software - The Software that interacts with the Hardware. example: OS
  • Application Software - The software that lets you interact with the system Software. types of app software
    • Browsers - The apps that let you interact with the internet. example: Chrome
    • Shareware - The apps that let you share data with other users. example: whatsapp
